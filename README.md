# Simulador-de-Colisoes-em-C
Simulador de colisões usando Raylib 
Para rodar o programa no code blocks, foram usados as seguintes configurações de compilador:
Linker Settings -> linked libraries:
raylib
opengl32
gdi32
winmm
Search Directories->Linker:
raylib-5.5_win64_mingw-w64\lib
Compiler Settings->Defines:
GRAPHICS_API_OPENGL 33
